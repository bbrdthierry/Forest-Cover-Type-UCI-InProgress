# Forest-Cover-Type-UCI
Academic project. The interest was to become familiar with machine learning tools on python, treating real unprocessed datas.

Subject : can we predict the forest cover type in the Roosevelt National Forest, based on some given cartographics informations ?

All the data used are in the train.csv file.
